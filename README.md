# Expense Tracker

## Description
A simple Python project that allows users to add, view, and delete expenses. It also calculates total spending.

## Features
- Add expenses with name and amount
- View all expenses
- Delete specific expenses
- Shows total spending

## Technologies Used
- Python
- File handling (txt file)

##  How to Run
1. Download the project
2. Run `expense tracker.py`
3. Follow menu options in terminal

##  Project Structure
expense-tracker/
- expense tracker.py
- expenses.txt
